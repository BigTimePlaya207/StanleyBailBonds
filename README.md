# Stanley Bail Bond LLC Website

A responsive business website built with React to showcase bail bond services and company information.

## Tech Stack
- React
- JavaScript (ES6+)
- CSS
- Git & GitHub

## Features
- Component-based React architecture
- Multi-page website structure
- Responsive layout for desktop and mobile
- Clean, maintainable UI components

## Run Locally
1. Clone the repository
2. Install dependencies: npm install
3. Start the application: npm start
